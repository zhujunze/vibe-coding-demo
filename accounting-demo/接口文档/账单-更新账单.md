# 更新账单

## 接口描述
更新账单记录

## 请求信息
- **URL**: `/api/transactions`
- **Method**: `PUT`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| id | Long | 是 | 账单ID | 1 |
| amount | BigDecimal | 否 | 金额 | 100.00 |
| categoryId | Long | 否 | 分类ID | 1 |
| type | String | 否 | 交易类型(EXPENSE/INCOME) | EXPENSE |
| date | String | 否 | 日期 | 2023-10-01 |
| note | String | 否 | 备注 | 餐饮 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": true,
  "trace_id": "c0a80101-..."
}
```

## 错误码

| 错误码 | 说明 |
|---|---|
| 400 | 账单不存在 |
| 403 | 无权限修改 |
