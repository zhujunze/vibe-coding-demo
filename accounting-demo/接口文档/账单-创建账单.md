# 创建账单

## 接口描述
新增一笔账单记录

## 请求信息
- **URL**: `/api/transactions`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| categoryId | Long | 是 | 分类ID | 1 |
| amount | BigDecimal | 是 | 金额 | 100.00 |
| type | String | 是 | 类型(expense/income) | expense |
| date | String | 是 | 日期 | 2023-10-01 |
| note | String | 否 | 备注 | 午餐 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": 1001,
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| data | Long | 账单ID | 1001 |

## 错误码

| 错误码 | 说明 |
|---|---|
| 400 | 参数校验失败（如分类不存在） |
| 403 | 无权使用该分类 |
