# 用户注册

## 接口描述
手机号 + 密码 + 短信验证码注册

## 请求信息
- **URL**: `/api/auth/register`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| phone | String | 是 | 手机号 | 13800138000 |
| password | String | 是 | 密码(>=8位，含大小写数字) | Password123 |
| smsCode | String | 是 | 短信验证码 | 123456 |
| inviteCode | String | 否 | 邀请码 | INVITE123 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": 1,
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| data | Long | 用户ID | 1 |

## 错误码

| 错误码 | 说明 |
|---|---|
| 20004 | 验证码错误 |
| 20005 | 手机号已存在 |
| 400 | 参数校验失败 |
