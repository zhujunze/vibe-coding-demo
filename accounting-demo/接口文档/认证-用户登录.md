# 用户登录

## 接口描述
支持手机号/邮箱 + 密码登录

## 请求信息
- **URL**: `/api/auth/login`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| account | String | 是 | 手机号/邮箱 | 13800138000 |
| password | String | 是 | 密码 | password123 |
| deviceInfo | String | 否 | 设备信息 | iPhone 14 Pro |
| captcha | String | 否 | 图形验证码 | AB12 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "userInfo": {
      "id": 1,
      "phone": "138****0000",
      "nickname": "用户123",
      "avatarUrl": "https://example.com/avatar.png",
      "createdAt": "2023-10-01 10:00:00"
    }
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| accessToken | String | 访问令牌 | eyJhbGciOiJIUzI1NiJ9... |
| refreshToken | String | 刷新令牌 | eyJhbGciOiJIUzI1NiJ9... |
| userInfo | Object | 用户基础信息 | - |
| userInfo.id | Long | 用户ID | 1 |
| userInfo.phone | String | 手机号(脱敏) | 138****0000 |
| userInfo.nickname | String | 昵称 | 用户123 |
| userInfo.avatarUrl | String | 头像URL | https://example.com/avatar.png |
| userInfo.createdAt | String | 注册时间 | 2023-10-01 10:00:00 |

## 错误码

| 错误码 | 说明 |
|---|---|
| 20002 | 用户不存在 |
| 20003 | 密码错误 |
| 401 | 账号已锁定 |
