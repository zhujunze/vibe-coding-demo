# 更新用户信息

## 接口描述
更新昵称、头像等信息

## 请求信息
- **URL**: `/api/users/me`
- **Method**: `PUT`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| nickname | String | 否 | 昵称 | 小明 |
| avatarUrl | String | 否 | 头像URL | https://example.com/avatar.png |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": true,
  "trace_id": "c0a80101-..."
}
```
