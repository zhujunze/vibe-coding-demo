# 获取分类占比

## 接口描述
获取指定时间范围内各分类的消费/收入占比

## 请求信息
- **URL**: `/api/charts/pie`
- **Method**: `GET`

## 请求参数

### Query Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| startDate | String | 否 | 开始日期 | 2023-10-01 |
| endDate | String | 否 | 结束日期 | 2023-10-31 |
| type | String | 否 | 统计类型(EXPENSE/INCOME) | EXPENSE |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "pieData": [
      {
        "name": "餐饮",
        "value": 500.00,
        "color": "#FF0000",
        "percentage": 0.25
      }
    ]
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| pieData | Array | 饼图数据 | - |
| name | String | 名称 | 餐饮 |
| value | BigDecimal | 金额 | 500.00 |
| color | String | 颜色 | #FF0000 |
| percentage | BigDecimal | 百分比 | 0.25 |
