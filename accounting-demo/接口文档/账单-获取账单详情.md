# 获取账单详情

## 接口描述
根据 ID 获取账单详情

## 请求信息
- **URL**: `/api/transactions/{id}`
- **Method**: `GET`

## 请求参数

### Path Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| id | Long | 是 | 账单ID | 1 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "categoryName": "餐饮",
    "categoryIcon": "food",
    "categoryColor": "#FF0000",
    "type": "支出",
    "amount": 100.00,
    "date": "2023-10-01",
    "note": "午餐",
    "categoryId": 10,
    "createdAt": "2023-10-01 12:00:00",
    "updatedAt": "2023-10-01 12:00:00"
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| id | Long | 账单ID | 1 |
| categoryName | String | 分类名称 | 餐饮 |
| categoryIcon | String | 分类图标 | food |
| categoryColor | String | 分类颜色 | #FF0000 |
| type | String | 交易类型 | 支出 |
| amount | BigDecimal | 金额 | 100.00 |
| date | String | 日期 | 2023-10-01 |
| note | String | 备注 | 午餐 |
| categoryId | Long | 分类ID | 10 |
| createdAt | String | 创建时间 | 2023-10-01 12:00:00 |
| updatedAt | String | 更新时间 | 2023-10-01 12:00:00 |

## 错误码

| 错误码 | 说明 |
|---|---|
| 400 | 账单不存在 |
| 403 | 无权限查看 |
