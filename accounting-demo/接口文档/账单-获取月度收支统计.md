# 获取月度收支统计

## 接口描述
查询指定月份的总收入、总支出和结余

## 请求信息
- **URL**: `/api/transactions/stats/month`
- **Method**: `GET`

## 请求参数

### Query Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| year | Integer | 是 | 年份 | 2023 |
| month | Integer | 是 | 月份 | 10 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "year": 2023,
    "month": 10,
    "totalIncome": 5000.00,
    "totalExpense": 3000.00,
    "balance": 2000.00
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| year | Integer | 年份 | 2023 |
| month | Integer | 月份 | 10 |
| totalIncome | BigDecimal | 总收入 | 5000.00 |
| totalExpense | BigDecimal | 总支出 | 3000.00 |
| balance | BigDecimal | 结余 | 2000.00 |
