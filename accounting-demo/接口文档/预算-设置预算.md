# 设置预算

## 接口描述
设置某分类在特定月份的预算

## 请求信息
- **URL**: `/api/budgets`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| categoryId | Long | 是 | 分类ID | 1 |
| month | String | 是 | 月份(YYYY-MM) | 2023-10 |
| amount | BigDecimal | 是 | 预算金额 | 2000.00 |
| alertThreshold | BigDecimal | 否 | 预警阈值 | 0.80 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": true,
  "trace_id": "c0a80101-..."
}
```
