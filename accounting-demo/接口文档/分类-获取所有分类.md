# 获取所有分类

## 接口描述
获取系统分类和用户自定义分类

## 请求信息
- **URL**: `/api/categories`
- **Method**: `GET`

## 请求参数

### Query Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| type | String | 否 | 分类类型(expense/income) | expense |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": [
    {
      "id": 1,
      "name": "餐饮",
      "type": "expense",
      "icon": "food",
      "color": "#FF0000",
      "isSystem": true
    }
  ],
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| id | Long | 分类ID | 1 |
| name | String | 分类名称 | 餐饮 |
| type | String | 分类类型 | expense |
| icon | String | 图标标识 | food |
| color | String | 颜色值 | #FF0000 |
| isSystem | Boolean | 是否系统预置 | true |
