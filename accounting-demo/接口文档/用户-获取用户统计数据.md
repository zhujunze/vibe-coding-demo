# 获取用户统计数据

## 接口描述
获取累计记账天数、打卡天数等统计信息

## 请求信息
- **URL**: `/api/users/stats`
- **Method**: `GET`

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "totalDays": 100,
    "totalRecords": 500,
    "continuousDays": 10
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| totalDays | Integer | 累计记账天数 | 100 |
| totalRecords | Integer | 记账总笔数 | 500 |
| continuousDays | Integer | 连续打卡天数 | 10 |
