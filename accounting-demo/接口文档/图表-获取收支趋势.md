# 获取收支趋势

## 接口描述
获取指定时间范围内的收入和支出趋势数据

## 请求信息
- **URL**: `/api/charts/trend`
- **Method**: `GET`

## 请求参数

### Query Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| startDate | String | 否 | 开始日期 | 2023-10-01 |
| endDate | String | 否 | 结束日期 | 2023-10-31 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "xAxis": ["10-01", "10-02"],
    "incomeData": [100.00, 0.00],
    "expenseData": [50.00, 20.00]
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| xAxis | Array | X轴标签(日期) | ["10-01", "10-02"] |
| incomeData | Array | 收入数据 | [100.00, 0.00] |
| expenseData | Array | 支出数据 | [50.00, 20.00] |
