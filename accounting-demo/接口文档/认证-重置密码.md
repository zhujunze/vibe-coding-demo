# 重置密码

## 接口描述
使用Token重置密码

## 请求信息
- **URL**: `/api/auth/reset-password`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| token | String | 是 | 重置Token | eyJhbGciOiJIUzI1NiJ9... |
| newPassword | String | 是 | 新密码 | NewPassword123 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": null,
  "trace_id": "c0a80101-..."
}
```

## 错误码

| 错误码 | 说明 |
|---|---|
| 401 | Token无效或过期 |
| 20002 | 用户不存在 |
| 400 | 新密码不能与旧密码相同 |
