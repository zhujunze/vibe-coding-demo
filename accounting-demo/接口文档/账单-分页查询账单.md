# 分页查询账单

## 接口描述
分页获取当前用户的账单列表，支持按时间、类型、分类筛选

## 请求信息
- **URL**: `/api/transactions`
- **Method**: `GET`

## 请求参数

### Query Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| page | Integer | 否 | 当前页码 (默认1) | 1 |
| size | Integer | 否 | 每页大小 (默认10) | 10 |
| year | Integer | 否 | 查询年份 | 2023 |
| month | Integer | 否 | 查询月份 | 10 |
| type | String | 否 | 交易类型(EXPENSE/INCOME) | EXPENSE |
| categoryId | Long | 否 | 分类ID | 1 |
| startDate | String | 否 | 开始日期 | 2023-10-01 |
| endDate | String | 否 | 结束日期 | 2023-10-31 |

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| records | Array | 记录列表 | - |
| id | Long | 账单ID | 1 |
| categoryName | String | 分类名称 | 餐饮 |
| categoryIcon | String | 分类图标 | food |
| categoryColor | String | 分类颜色 | #FF0000 |
| type | String | 交易类型 | 支出 |
| amount | BigDecimal | 金额 | 100.00 |
| date | String | 日期 | 2023-10-01 |
| note | String | 备注 | 午餐 |
| total | Integer | 总记录数 | 100 |
| size | Integer | 每页大小 | 10 |
| current | Integer | 当前页码 | 1 |
| pages | Integer | 总页数 | 10 |
