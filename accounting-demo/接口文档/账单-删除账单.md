# 删除账单

## 接口描述
根据 ID 删除账单

## 请求信息
- **URL**: `/api/transactions/{id}`
- **Method**: `DELETE`

## 请求参数

### Path Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| id | Long | 是 | 账单ID | 1 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": true,
  "trace_id": "c0a80101-..."
}
```

## 错误码

| 错误码 | 说明 |
|---|---|
| 403 | 无权限删除 |
