# 创建自定义分类

## 接口描述
用户添加新的账单分类

## 请求信息
- **URL**: `/api/categories/custom`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| name | String | 是 | 分类名称 | 游戏 |
| type | String | 是 | 类型(expense/income) | expense |
| icon | String | 是 | 图标 | game |
| color | String | 是 | 颜色 | #000000 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": 10,
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| data | Long | 分类ID | 10 |

## 错误码

| 错误码 | 说明 |
|---|---|
| 400 | 分类名称已存在 |
