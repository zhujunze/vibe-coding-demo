# 刷新令牌

## 接口描述
使用刷新令牌获取新的访问令牌。当 Access Token 过期时，使用 Refresh Token 换取新的 Access Token 和 Refresh Token。

## 请求信息
- **URL**: `/api/auth/refresh-token`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| refreshToken | String | 是 | 刷新令牌 | eyJhbGciOiJIUzI1NiJ9... |

## 错误码

| 错误码 | 说明 |
|---|---|
| 403 | 无效的刷新令牌 |
| 20002 | 用户不存在 |
