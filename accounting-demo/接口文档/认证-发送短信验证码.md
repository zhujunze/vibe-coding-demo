# 发送短信验证码

## 接口描述
向指定手机号发送验证码

## 请求信息
- **URL**: `/api/auth/send-sms`
- **Method**: `POST`
- **Content-Type**: `application/json`

## 请求参数

### Body Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| phone | String | 是 | 手机号 | 13800138000 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": null,
  "trace_id": "c0a80101-..."
}
```

## 错误码

| 错误码 | 说明 |
|---|---|
| 400 | 手机号格式错误 |
| 429 | 请求过于频繁 |
