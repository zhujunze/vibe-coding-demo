# 删除自定义分类

## 接口描述
删除用户自定义的分类

## 请求信息
- **URL**: `/api/categories/{id}`
- **Method**: `DELETE`

## 请求参数

### Path Parameters

| 参数名 | 类型 | 必填 | 说明 | 示例 |
|---|---|---|---|---|
| id | Long | 是 | 分类ID | 1 |

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": true,
  "trace_id": "c0a80101-..."
}
```

## 错误码

| 错误码 | 说明 |
|---|---|
| 403 | 无法删除系统分类或他人分类 |
| 400 | 分类不存在 |
