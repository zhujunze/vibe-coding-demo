# 获取当前用户信息

## 接口描述
获取登录用户的基本信息

## 请求信息
- **URL**: `/api/users/me`
- **Method**: `GET`

## 响应结构

### 成功响应 (200 OK)

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "phone": "138****0000",
    "nickname": "用户123",
    "avatarUrl": "https://example.com/avatar.png",
    "createdAt": "2023-10-01 10:00:00"
  },
  "trace_id": "c0a80101-..."
}
```

### 数据字段说明 (Data Fields)

| 字段名 | 类型 | 说明 | 示例 |
|---|---|---|---|
| id | Long | 用户ID | 1 |
| phone | String | 手机号(脱敏) | 138****0000 |
| nickname | String | 昵称 | 用户123 |
| avatarUrl | String | 头像URL | https://example.com/avatar.png |
| createdAt | String | 注册时间 | 2023-10-01 10:00:00 |
